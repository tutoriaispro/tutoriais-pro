🔴 Erro VAN 9005 / VAN 9003 / VAN 9002

🎮 VALORANT · Anti-Cheat Vanguard

O erro VAN9005 é causado quando os requisitos de segurança exigidos pela Riot Games não estão ativados corretamente — principalmente o TPM 2.0 e o Secure Boot.

Causas mais comuns:

TPM 2.0 DESABILITADO SECURE BOOT DESABILITADO

Para a maioria dos casos, habilitar essas configurações resolve o problema. O processo é feito pela BIOS do seu computador:

  • Pressione o botão do Windows e escreva "Informações", abra "Informações do Sistema"
  • Localize a opção "Produto BaseBoard" — ela mostrará o nome da sua placa-mãe
  • Com o nome da placa-mãe em mãos, pesquise no YouTube ou ChatGPT:
    • "Como resolver VAN9005 Valorant (nome da placa-mãe)"
    • "Como ligar Secure Boot e TPM 2.0 na BIOS (nome da placa-mãe)"
💡 Dica: Pesquisar no YouTube geralmente é mais eficaz, pois existem vídeos com passo a passo gravado para cada placa-mãe específica. Veja o tutorial de TPM 2.0 detalhado abaixo para Intel e AMD.

Referência de vídeo: TPM 2.0 & Secure Boot — Erro VAN 9005/9003/9002 (YouTube)

🔵 Como Ativar TPM 2.0 — Intel & AMD

O TPM 2.0 é um chip de segurança obrigatório para Valorant e alguns outros jogos. Siga o guia específico para o seu processador:

🔵 Intel
🔴 AMD

🔵 Ativando TPM 2.0 — Processadores Intel

Em placas Intel, o TPM é chamado de Intel PTT (Platform Trust Technology).

1 Acesse a BIOS/UEFI
  • Desligue completamente o computador
  • Ligue e pressione repetidamente a tecla Delete ou F2 durante a inicialização
  • Algumas placas ASUS: F2 | Gigabyte: Delete | MSI: Delete | ASRock: F2
2 Acesse o Modo Avançado
  • Na BIOS, pressione F7 ou procure o botão "Advanced Mode"
  • Isso exibe todas as opções detalhadas da placa
3 Encontre o Intel PTT
  • Navegue até a aba Advanced → Trusted Computing
  • Em algumas placas: Security → TPM Configuration
  • Em outras: Settings → Security → Trusted Computing
  • Encontre a opção Security Device Support e mude para Enable
  • Encontre TPM Device Selection ou TPM Support e selecione Intel PTT
4 Salve e Reinicie
  • Pressione F10 para salvar
  • Confirme clicando "OK" ou "Yes"
  • O computador reiniciará automaticamente
5 Verifique no Windows
  • Pressione Win + R, digite tpm.msc e pressione Enter
  • Deverá aparecer: "O TPM está pronto para uso" e Versão da especificação: 2.0
✅ Intel PTT ativado! Agora o Valorant deve conseguir iniciar normalmente.
⚠️ Não encontra Intel PTT? Atualize o firmware (BIOS) da sua placa-mãe para a versão mais recente. Placas muito antigas podem não suportar TPM 2.0.

🔴 Ativando TPM 2.0 — Processadores AMD

Em placas AMD, o TPM é chamado de AMD fTPM (Firmware TPM) ou AMD PSP fTPM.

1 Acesse a BIOS/UEFI
  • Desligue completamente o computador
  • Ligue e pressione repetidamente Delete ou F2
  • Placas AMD mais comuns: ASUS: F2/Delete | Gigabyte: Delete | MSI: Delete
2 Entre no Modo Avançado
  • Pressione F7 ou clique em "Advanced Mode"
  • Procure a aba Advanced ou Settings
3 Encontre o AMD fTPM
  • Navegue até Advanced → AMD fTPM configuration
  • Em algumas placas: Advanced → Trusted Computing
  • Em outras: Settings → Security → AMD PSP fTPM
  • Encontre a opção TPM Device Selection ou AMD CPU fTPM
  • Mude de Discrete TPM ou Disabled para Firmware TPM ou AMD CPU fTPM
4 Salve e Reinicie
  • Pressione F10 para salvar
  • Confirme com "OK" ou "Yes"
  • O sistema reiniciará
5 Verifique no Windows
  • Pressione Win + R, digite tpm.msc e pressione Enter
  • Deve aparecer: "O TPM está pronto para uso" — Versão: 2.0
✅ AMD fTPM ativado! Agora o Valorant deve conseguir iniciar sem o erro VAN9005.
⚠️ Travamentos com AMD fTPM? Alguns processadores Ryzen 1000/2000/3000 podem ter instabilidade com fTPM. Se ocorrer travamentos no Windows após habilitar, atualize o AGESA da BIOS ou pesquise "AMD fTPM stutter fix (modelo do seu Ryzen)".

🟡 Fix Secure Boot Enabled

🎮 VALORANT · Erro VAN9003

O Secure Boot é um recurso de segurança da UEFI que precisa estar ativo para o Vanguard (anti-cheat do Valorant) funcionar corretamente. Siga os passos abaixo:

👑 Passo a Passo para Configurar o Secure Boot

1 Acesse a BIOS/UEFI
  • Desligue seu computador completamente
  • Ligue novamente e pressione repetidamente a tecla Delete (Del) enquanto o computador inicia
  • Alternativas: Algumas placas-mãe utilizam outras teclas como F2, F10, ou Esc — consulte o manual da sua placa-mãe
2 Localize o Secure Boot
  • Navegue até a aba Boot, Tools, ou Power, dependendo da sua placa-mãe
  • Se Secure Boot não estiver visível:
    • Vá até a aba Boot
    • Desative a opção CSM (Compatibility Support Module)
    • Salve as configurações e reinicie para retornar à BIOS
    • Agora o Secure Boot estará disponível
3 Configurar o Secure Boot
  • Encontre a opção Secure Boot e ative (Enabled)
  • Realize os seguintes ajustes:
    • Reset Keys: Resete as chaves do Secure Boot
    • Restore Factory Keys: Retorne para as chaves de fábrica
    • Se a opção estiver configurada como Standard, altere para Custom — isso permitirá o uso de botões para gerenciar as chaves
4 Salve e Reinicie

Após configurar o Secure Boot, salve as alterações (geralmente pressionando F10) e reinicie o computador.

✅ Resolvido! Após ativar o Secure Boot com as chaves corretas, o Valorant deverá iniciar sem o erro VAN9003.
⚠️ Atenção: Se o Windows não iniciar após ativar o Secure Boot, entre novamente na BIOS e faça o "Restore Factory Keys" antes de ativar. Sistemas com dual boot podem exigir configurações adicionais.

🟠 Fix HVCI Enabled

🎮 VALORANT · Erro VAN:RESTRICTION

O erro "Your account does not meet the following requirements: HVCI enabled" significa que a Integridade de Memória (HVCI) precisa estar ativada no Windows.

Como Ativar a Integridade de Memória

1 Abra o Isolamento de Núcleo
  • Aperte o botão Windows do seu computador
  • Escreva "isolamento de núcleo" na barra de pesquisa
  • Abra o resultado que aparecer
2 Ative a Integridade de Memória
  • Encontre a opção "Integridade de memória"
  • Ative o toggle (mude para Ligado)
3 Reinicie o Computador
  • Reinicie o computador para aplicar as alterações
  • Após reiniciar, abra o Valorant normalmente
✅ Resolvido! Após habilitar a Integridade de Memória e reiniciar, você poderá abrir o VALORANT novamente.
⚠️ HVCI Incompatível com drivers? Se a opção estiver cinza ou não puder ser ativada, pode haver um driver incompatível no sistema. Verifique o Gerenciador de Dispositivos por drivers com erros ou atualize todos os drivers antes de tentar novamente.

🟢 Conexão Falhada? Chrome ou Edge

Se você está tentando acessar o valorantskinchanger.com.br/app e a conexão está falhando no Chrome ou Edge, siga os passos abaixo:

Como Resolver

1 Acesse as Configurações do Site
  • Com o site aberto no navegador, clique no ícone de cadeado (ou na engrenagem) ao lado da URL
  • Um menu de permissões do site irá abrir
2 Ative o Acesso à Rede Local
  • Localize a opção "Acesso à rede local" no menu
  • Verifique se essa opção está ativada
  • Caso esteja desativada, habilite-a
3 Recarregue a Página
  • Após ativar, recarregue a página com F5 ou Ctrl+R
✅ Pronto! Após ativar o acesso à rede local, o problema de conexão deverá ser resolvido.
💡 Nota: Essa opção controla se o site pode se comunicar com servidores locais da sua máquina (como o backend do SkinChanger que roda localmente). Sem ela ativada, o site não consegue se conectar ao processo rodando no seu PC.

⚪ SkinChanger Crashando Após Injetar

Se o SkinChanger está crashando após a injeção no game, o problema geralmente é o Windows Defender bloqueando o processo. Resolva adicionando uma exclusão:

1 Abra a Proteção contra Vírus
  • Pressione o botão Windows e pesquise: "Proteção contra vírus e ameaças"
  • Abra o resultado
2 Acesse as Configurações de Proteção
  • Desça a página e encontre a opção "Configurações de proteção contra vírus e ameaças"
  • Clique em "Gerenciar configurações"
3 Adicione as Exclusões
  • Desça até a opção "Adicionar ou remover exclusões" e clique nela
  • Remova o SkinChanger e o WebDoc das exclusões (se já estiverem lá) e readicione-os
  • Adicione tanto a pasta do SkinChanger quanto o executável como exclusões separadas
4 Reinicie e Tente Novamente
  • Feche o jogo completamente
  • Reabra o SkinChanger e tente injetar novamente
✅ Resolvido! Com as exclusões configuradas corretamente, o SkinChanger funcionará sem crashar.
💡 Ainda está crashando? Se mesmo após adicionar as exclusões o game continuar crashando, abra um ticket de suporte no canal #desconhecido que a equipe ajudará a resolver da melhor forma.

❌ Aplicativos Incompatíveis / Proibidos

Certifique-se de que os seguintes aplicativos estejam completamente desinstalados antes de usar o SkinChanger ou jogar Valorant. Eles podem causar conflitos, crashs ou banimento:

GlassWire
Everything
NetLimiter
HTTPDebugger
Wireshark
Npcap
Fiddler
FiddlerEverywhere
BinaryNinja
ProcessHacker
OverWolf
CheatEngine
ProcessLasso
Lunar Client
Bad Lion Client
⚠️ Importante: Não basta fechar esses programas — eles devem estar desinstalados completamente. Alguns deles deixam drivers rodando em segundo plano mesmo quando fechados.

Como Desinstalar Completamente

  • Acesse Painel de Controle → Programas → Programas e Recursos
  • Localize o programa na lista e clique em Desinstalar
  • Após desinstalar, reinicie o computador para garantir que nenhum driver residual permaneça ativo
  • Para verificar drivers persistentes, use o Gerenciador de Dispositivos ou o Autoruns (da Microsoft Sysinternals)
💡 Dica extra: Programas como Npcap e WinPcap podem permanecer como drivers mesmo após desinstalar o software principal. Pesquise especificamente por "como remover Npcap completamente" se precisar.
