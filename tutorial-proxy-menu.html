📌 Introdução

Bem-vindo ao tutorial completo do Proxy Menu para FiveM! Este mod menu oferece funcionalidades avançadas (aimbot, ESP, noclip, teleporte, spawn, etc.). O acesso é exclusivo para clientes no servidor oficial do Discord.

Antes de tudo, instale corretamente o **Microsoft Visual C++ Redistributable** (pacote All-in-One oficial) e o **Sordum Defender Control** para evitar erros de execução, DLLs faltando ou bloqueios de antivírus.

🛡️ Aviso Importante – Fevereiro 2026

⚠️ Responsabilidade: O Proxy Menu viola os Termos de Serviço da Rockstar, Cfx.re e FiveM. Pode resultar em ban permanente (HWID/IP/global). Use apenas em servidores privados com permissão e por sua conta e risco.

🔧 Requisitos e Preparação Obrigatória

1. Instalar Microsoft Visual C++ Redistributable (All-in-One)

Este pacote é essencial para rodar o loader sem erros de DLLs faltando (ex: MSVCR140.dll, VCRUNTIME140.dll). Baixe a versão oficial All-in-One do TechPowerUp (atualizada e segura):

Baixar Visual C++ All-in-One (TechPowerUp)

  1. Acesse o link acima e clique em "Download" (escolha a versão mais recente, geralmente ~50-60 MB).
  2. Salve o arquivo (ex: VisualCppRedist_AIO_x86_x64.exe).
  3. Clique direito no arquivo → "Executar como administrador".
  4. Na janela que abrir, marque a opção "Install All" ou "Instalar Todos" (inclui 2005 a 2022 x86 e x64).
  5. Clique em "Install" e aguarde a conclusão (pode levar 2-5 minutos).
  6. Reinicie o PC quando solicitado (obrigatório para aplicar as mudanças).
✅ Concluído! O C++ está instalado e o loader do Proxy Menu deve rodar sem erros de bibliotecas.

2. Instalar e Configurar Sordum Defender Control

Essa ferramenta desativa temporariamente o Windows Defender (proteção em tempo real e isolamento de núcleo) para evitar bloqueios no loader.

Baixar Sordum Defender Control (Oficial)

  1. Baixe o arquivo (DefenderControl.zip ou .exe).
  2. Extraia o ZIP (se for o caso) usando 7-Zip ou WinRAR.
  3. Clique direito em DefenderControl.exe → "Executar como administrador".
  4. Na janela simples que abre:
    • Clique no botão "Disable" ou "Desativar" (para proteção em tempo real).
    • Marque também "Disable Core Isolation / Memory Integrity" se disponível (evita erro de isolamento de núcleo).
  5. Confirme com "Yes" ou "Sim" nas mensagens do UAC.
  6. Feche o programa (as configurações permanecem até você reativar).
Importante: Reative tudo após usar o Proxy Menu (clique "Enable" no Sordum e reinicie o PC).

Passo a Passo: Acesso e Uso do Proxy Menu

1 Entrar no Servidor Oficial (Exclusivo para Clientes)

O download do loader e chave são liberados apenas para membros do servidor oficial:

Entrar no Servidor Oficial do Proxy Menu

Após entrar, vá ao canal de downloads ou abra ticket para receber o loader e sua chave (exclusivo para clientes pagos).
2 Cadastro e Login no Loader
  1. Execute o loader como administrador.
  2. Registre-se (crie usuário/senha) → clique "Register".
  3. Faça login (marque "Remember") → clique "Log In".
3 Injeção no FiveM
  1. Abra o FiveM.
  2. No loader, clique no ícone "Prox" → depois no botão de injeção ("Prox Executor").
  3. Aguarde conclusão (menu aparece no jogo).

🐛 Troubleshooting

  • Erro de DLLs faltando: Reinstale Visual C++ All-in-One e reinicie o PC.
  • Loader bloqueado: Use Sordum para desativar Defender corretamente.
  • Injeção falha: Abra FiveM primeiro, verifique chave válida no servidor oficial.
